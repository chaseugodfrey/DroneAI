Press 
F1 for 3D Pathfinding Demo
F2 for Nav Mesh Demo (obsolete)
F3 for Drone AI Demo

----------------------------------------------

3D Pathfinding Demo

Controls:

Camera Control:
When using the scene camera
-> Hold shift for all controls
-> Shift + WASD to move front back left right
-> Shift + E/Q to ascend and descend
-> Shift + HOLD right click to rotate camera

Right Click to choose a destination for plane agent (in non-edit mode)

UI:

Swap Camera:
Swaps POV between scene camera and agent camera.

Auto-pilot:
Agent automatically chooses destinations without needing to click on blocks.

Edit Data
-> left click to add
-> right click to remove
base blocks cannot be removed

Create Random Generated Blocks
-> Randomly creates blocks in the map

Clear map
-> Removes all blocks except base

Notes:
Save Data might not work as intended.

Drone AI Demo

Controls:
WASD movement
Shift -> Sprint
Mouse Input -> Rotate Camera

Z -> Settings
X -> Toggle Player Occupancy Map
C -> Toggle Enemy Visibility
V -> Toggle Enemy Occupancy Map
B -> Toggle Enemy Flow Field
